# Serrano
Serrano is a [hypermedia](https://en.wikipedia.org/wiki/Hypermedia) implementation for [Avocado](https://github.com/cbmi/avocado/wiki) that adheres to the constraints of [REST](https://en.wikipedia.org/wiki/Representational_state_transfer).

Don't know anything about Avocado? [Read about it](https://github.com/cbmi/avocado/wiki).

Read the docs on the [wiki](https://github.com/cbmi/serrano/wiki)
